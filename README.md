# evalfinal_GesTareaV5
Aplicación Android/Kotlin que permite al usuario registrar sus actividades diarias (nombre de la actividad, fecha, hora y descripción) y visualizar el historial de registros. La app aplica y gestiona el ciclo de vida de los componentes y utiliza una arquitectura MVVM, Coroutines; Fragments, ViewModel, LiveData, Companion, Factory, Jetpack.
